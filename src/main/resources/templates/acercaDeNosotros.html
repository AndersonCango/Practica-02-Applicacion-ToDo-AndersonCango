Aplicación de Gestión de Tareas

Anderson Daniel Cango Samuisa

Metodologías Ágiles - Ingeniería en Software

Esta aplicación fue desarrollada como parte del proyecto académico en la Escuela Politécnica Nacional, utilizando Spring Boot como framework principal y aplicando metodologías ágiles para su planificación, implementación y mejora continua.

Su objetivo es ofrecer una herramienta eficiente y moderna para la gestión de tareas, facilitando el control de actividades, seguimiento de progreso y colaboración entre usuarios. Esta solución está diseñada para ser intuitiva, adaptable y escalable en entornos académicos y profesionales.
